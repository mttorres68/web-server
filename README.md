# Como rodar o projeto ✅
### Para baixar o projeto, clone-o diretamente do git executando o seguinte comando
```
git clone https://github.com/mttorres68/web-server.git
```

### Instale as dependências 
```
npm install
```

#### Rodar o projeto
```
npm run dev
```
